= imitation

random input generator for testing

== Usage

  class Data
    extend Imitation
  
    AUTHORS = %w[ okitan kuny okitakunio ]
  end

  Data.author  #=> okitan, kuny or okitakunio
  Data.author? #=> okitan, kuny, okitakunio or ''
  Data.authors #=> e.g. %w[ okitan ], %w[ kuny okitakunio ]
